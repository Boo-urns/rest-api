rest-api
========

REST API for Meteor, using Iron Router.  You can find an online demo at http://rest-api.meteor.com/  

Installation
------------------------


````sh
# Should be as simple as cloning the repository...  
git clone https://github.com/awatson1978/rest-api.git

# And then running it...
cd rest-api
meteor
````
