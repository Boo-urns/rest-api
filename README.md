rest-api
========

REST API for Meteor, using Iron Router.  See the [rest-api demo app](https://github.com/awatson1978/rest-api).  

Installation
------------------------


````sh
# Should be as simple as cloning the repository...  
git clone https://github.com/awatson1978/rest-api.git

# And then running it...
cd rest-api
meteor
````
